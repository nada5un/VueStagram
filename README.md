### 📌 주요기능  

- 게시글 작성 가능 
- 인스타그램 필터 적용 가능 
- 더보기 버튼을 통한 외부 데이터 요청 가능 
- 팔로워 페이지 실시간 검색 가능 
- 게시글 좋아요 및 좋아요 취소 가능
